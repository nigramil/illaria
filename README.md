# Weather Diary

**Автор:** Иллария Гаянова

## Описание
Приложение "Weather Diary" позволяет добавлять записи о погоде, фильтровать их по дате и температуре, сохранять в JSON файл и загружать при необходимости.

## Как запустить
1. Убедитесь, что у вас установлен Python 3.
2. Скачайте проект.
3. Запустите файл `weather_diary.py` командой:
```bash
python weather_diary.py
